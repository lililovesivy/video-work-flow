# 样片构图规律：长文本、原素材、重点提示、人物

## 核心判断

这组样片的构图逻辑不是把信息重新设计成精美信息图，而是把画面做成“普法内容页”：原素材保留证据感，重点文字负责解释，人物或标签负责降低理解成本。

优先级：

1. 原素材可信度
2. 重点结论可读性
3. 观众阅读顺序
4. 人物或示意图辅助理解
5. 画面装饰

## 画面基础

- 画面主体仍按 `16:9` 横向内容区设计。
- 背景使用浅灰、米白、纸纹、网点、扫描纹理，避免纯白空板。
- 颜色控制在黑、灰、白、黄、红五类：黑字讲事实，黄标划重点，红字给结论。
- 不追求对齐很“PPT”，允许截图、文件、人物有轻微倾斜和真实素材拼贴感。

## 长文本处理

长文本不要整段重排成设计文字。正确做法是把长文本当作“证据原件”处理。

- 如果长文本来自法条、判决、新闻、合同，优先以截图/文件页形式出现。
- 原文可以不全部读清，但必须让观众看出“这是原始材料”。
- 真正需要观众读懂的内容，只放大或高亮 `1-3` 句。
- 长文本截图可以裁切、倾斜、局部放大，形成证据页质感。
- 页面上另配一条短解释或结论，替代让观众读完整段原文。

推荐结构：

- 左侧大标题 + 一句解释，右侧放倾斜文件截图，并在文件中高亮关键句。
- 中间放文件截图，右侧用 `2-3` 条黄标提示提炼判断点。
- 下方用红色大字给结论，截图只作为上方证据支撑。

避免：

- 把长文本全部转成多个同等权重卡片。
- 一屏超过 `4` 个长句说明。
- 截图、标题、解释文字都同样大，导致没有阅读顺序。

## 原素材截图处理

原素材是主角，不是小装饰。尤其是新闻截图、法条截图、判决截图、网页截图。

- 有原素材时，截图通常占画面 `45%-65%`。
- 截图可以做大幅裁切，只保留和口播相关的部分。
- 重点段落用黄色荧光笔、下划线、红框，不要用复杂图标替代。
- 截图旁边只放短结论，通常 `8-16` 个字。
- 如果截图文字太小，就截取关键区域放大，不要硬塞完整页面。

推荐构图：

- `证据主图 + 旁侧短结论`：截图占左/中，右侧黄标或黑字短句。
- `原文页 + 标题解释`：左侧标题解释，右侧文件倾斜铺开。
- `证据页 + 红色结论`：上方证据截图，下方红色大字判断。

避免：

- 把新闻/法条截图缩成小缩略图。
- 在截图上压大段解释文字。
- 用示意图抢走新闻截图或法条截图的主视觉。

## 重点提示文字

重点提示不是普通标签，而是画面阅读路线的“路标”。

- 黑色大标题负责抛出问题或概念。
- 黄色高亮负责锁定关键词、判断条件、资金路径、比例数字。
- 红色大字只用于最终判断、风险结论、反常点。
- 黄标文字要短，最好 `4-12` 个字。
- 右侧可以竖向排列 `2-3` 条黄标，形成清单感。
- 底部红字或黑底白字结论只能放一条，不要和主标题抢层级。

常用组合：

- 大黑标题：`诈骗到底怎么认定？`
- 右侧黄标：`资金路径是否清晰`
- 底部红字：`判断不看承诺 看三点`

避免：

- 每个说明都做成同样大小的黄色标签。
- 黄色标签压住人物、截图或箭头。
- 红色结论过多，导致全画面都像重点。

## 人物构图

人物在这类样片里通常是“角色锚点”，不是完整故事场景。

- 人物可居中，占画面高度 `60%-85%`，左右留出标签区。
- 人物可以是真实抠图、背影、剪影、模糊脸、遮脸，重点是性别、年龄、身份能被读出来。
- 人物边缘可加极细黄色描边或弱发光，用来从浅色背景中分离。
- 标签放在人物左右空白处，用虚线或细线指向身体，不压在人物身上。
- 单个人物页通常只放 `2-3` 个标签。
- 标签框用白底黑边，文字黑色，不要做花哨色块。

推荐结构：

- `人物居中 + 左右标签`：人物是主体，标签解释心理、行为、风险点。
- `人物侧放 + 文件/证据主图`：人物只占一侧，另一侧放核心证据。
- `人物半身 + 大标题`：适合表达当事人状态或口播里的角色引入。

避免：

- 人物遮挡核心标题或原素材截图。
- 人物太小，只剩装饰作用。
- 标签线穿过人物脸部或关键文字。

## 四种常用版式

### 1. 长文本法条页

- 左侧：大标题 + 一句解释。
- 右侧：倾斜法条/判决/文件截图。
- 文件中：只高亮关键 `1-3` 行。
- 下方：可加一句字幕或短结论。

适合：法条解释、判决依据、规则条件。

### 2. 证据判断页

- 上方或左侧：证据截图、流程图或资金路径。
- 右侧：`2-3` 条黄色判断条件。
- 下方：红色结论。
- 箭头只连接空白区域，不穿过文字和截图。

适合：资金流、诈骗认定、产权流转、证据链。

### 3. 人物标签页

- 中间：人物抠图或剪影。
- 左右：白底黑边标签。
- 标签线：细虚线，短且不穿过主体。
- 背景：浅灰纸纹或网点。

适合：心理状态、角色画像、行为动机、风险身份。

### 4. 原素材主导页

- 中间或左侧：新闻/法条/截图作为最大元素。
- 下方或右侧：示意图缩小辅助。
- 重点文字只解释截图，不替代截图。
- 结论用一条黄标或红字，不额外堆卡片。

适合：新闻截图、案件截图、官方文件、网页页面。

## 动画对应规则

- 原素材截图：轻微推近，关键句黄标扫过。
- 长文本文件：从右侧或下方缓慢滑入，随后高亮关键句。
- 重点提示：按口播节奏逐条出现，不要同时出现。
- 人物：轻微淡入或推近，标签随后依次贴出。
- 结论：最后用红字压入或黑底白字条出现。

## 生成前检查

每张预览图和每个视频关键帧必须检查：

- 原素材是否是主角，示意图是否没有抢戏。
- 长文本是否只突出关键句，没有让观众读整页。
- 黄标和红字是否少而明确。
- 人物是否没有遮挡核心文字或截图。
- 标签线是否没有穿过人物脸部、关键字、截图高亮区。
- 所有文字、箭头、标签、人物、截图之间是否没有遮挡。
