# 法律名词 UI 使用规则

## 固定背景

以后生成预览图或视频素材时，默认使用这张 16:9 背景作为基础背景：

`/Users/lili/Desktop/视频素材工作流/参考素材/背景/法律名词ui-通用背景-16x9.png`

这张背景由 `法律名词ui/背景图.png` 抠图后扩展生成，不直接拉伸原 400×400 原图。

## 抠图缓存

原始 UI 图片目录：

`/Users/lili/Desktop/视频素材工作流/法律名词ui`

抠图缓存目录：

`/Users/lili/Desktop/视频素材工作流/已抠图ui`

规则：

- 不直接使用 `法律名词ui` 里的原图。
- 使用前先检查 `已抠图ui` 是否已有同名图片。
- 如果已有同名抠图，直接使用抠图缓存。
- 如果没有同名抠图，先抠图并保存到 `已抠图ui`。
- 抠图文件必须和原图同名，例如原图 `占有.png`，抠图也保存为 `已抠图ui/占有.png`。
- 背景图抠图固定保存为 `已抠图ui/背景图.png`。

## 处理脚本

通用处理脚本：

`/Users/lili/Desktop/视频素材工作流/生成脚本/process_legal_ui_assets.py`

默认运行时处理 `背景图.png`，并生成 16:9 通用背景。

如需按需抠某个 UI 图，传入文件名：

```bash
python3 /Users/lili/Desktop/视频素材工作流/生成脚本/process_legal_ui_assets.py 占有.png
```

## 使用原则

- UI 图只作为法律概念、关键词、名词解释的辅助视觉元素。
- 有新闻截图、法条截图、判决截图时，原素材仍是主视觉，UI 图不能抢主体。
- UI 图应放在空白区或作为小型辅助图标，不压文字、不压截图、不压人物。
- 如果 UI 图颜色和当前画面不协调，降低透明度或缩小比例，优先保证证据和文字可读。

## 生成前检查

每次使用法律名词 UI 图时必须检查：

- 是否使用 `已抠图ui` 的抠图版本，而不是原图。
- 抠图是否和原图同名。
- 16:9 背景是否没有拉伸变形。
- UI 图是否没有遮挡标题、证据截图、人物、箭头、标签或底部结论。
