# 视频素材生成工作流

这是一个面向法律、商业、财经类口播视频的素材生成工作流。

目标不是一键生成整条视频，而是把口播文稿拆成可审核、可批量生成、可插入主时间线的独立短视频素材。

## 核心规则

- 逐句拆解口播文稿，先生成文稿和视觉素材对照表。
- 每个视觉素材编号对应一个独立 MP4，不能把多个素材拼在一个文件里。
- 每批只推进 2 个素材。
- 先生成静态预览图，用户确认后再生成视频。
- 生成后必须自检遮挡、裁切、错字、变形、图文对应关系。
- 有新闻截图、法条截图、判决截图时，原素材优先作为主视觉。
- 动画按主题、主体、关系、标签、证据强调、结论依次出现。
