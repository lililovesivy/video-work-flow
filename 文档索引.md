# 视频素材工作流文档索引

更新时间：2026-06-17

## 先读哪份文档

如果只读一份，先读：

- [说明文档/视频素材生成完整工作流.md](说明文档/视频素材生成完整工作流.md)

这份文档已经整合了最近一轮工作流变更，包括：

- 从 `2` 个素材一批改成 `4` 个素材一批
- 从 `PNG` 预览审核改成 `PPT` 审核优先
- 从单 agent 串行改成 `2` 个 sub-agent 并行
- 从“一页尽量装满”改成“一页一个阅读任务”
- 新增“禁止内部编号、禁止制作提示语、色块少而准”等规则

## 当前主结论

当前正式生产链路固定为：

**逐句视觉审核表 -> 4 页可编辑 PPT 并行生成 -> 用户修改/确认 PPT -> 直接生成视频 -> 下一批 4 页**
